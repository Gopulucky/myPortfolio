# Professional Software Engineer Portfolio

A modern, responsive, and high-performance portfolio website designed for a Software Engineer. Built with semantic HTML5, CSS3 (Glassmorphism), and Vanilla JavaScript.

## Features

-   **Professional Theme**: Navy and Slate Blue color palette with Gold/Cyan accents.
-   **Responsive Design**: Fully responsive layout that works on Mobile, Tablet, and Desktop.
-   **Interactivity**:
    -   Smooth scrolling for navigation.
    -   Mobile toggle menu.
    -   Fade-in scroll animations.
-   **Sections**:
    -   **Hero**: Strong value proposition and call to action.
    -   **About**: Concise professional summary and tech stack visualization.
    -   **Experience**: Timeline of work history.
    -   **Work**: Featured projects with "folder" style cards.
    -   **Contact**: Clean and inviting contact section.

## Tech Stack

-   **HTML5**
-   **CSS3** (Custom Properties, Flexbox, Grid)
-   **JavaScript** (ES6+)
-   **Font Awesome** (Icons)
-   **Google Fonts** (Inter & Playfair Display)

## How to Run

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/portfolio.git
    ```
2.  Open `index.html` in your browser.

## customization

1.  Update `index.html` with your specific project details, links, and contact info.
2.  Replace the placeholder text in the "Experience" section.
3.  (Optional) Add your resume PDF to the root directory and link it in the "Contact" section.
